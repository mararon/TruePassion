TruePassion is a school project created as a part of the course 
"DV1486, Databasdrivna webbapplikationer med PHP och MVC-ramverk" 
at Blekinge Tekniska Högskola with the purpose to learn how to 
construct an MVC framework.
 
TruePassion is based on Lydia https://github.com/mosbth/lydia
